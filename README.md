# Mind-mail
An AI-driven smart email **SAAS**. 

## Purpose
Try to utilize AI and modern user-friendly development approaches to solve pain points of using email app.

## Stack
This is a [T3 Stack](https://create.t3.gg/) project bootstrapped with `create-t3-app`. Specified stacks are:
- `Next.js` as full-stack development
- `Prisma` as Database ORM
- `PostgreSQL` as Database
- `Taliwind` for app styles
- `TRPC`for type security

### Externel Services
- **Clerk** authentication
- **Aurinko** for email sync
- **Neon.tech** for ***serverless db***
- **shadcn** for fast UI building
- **Vercel AI SDK** for RAG pipeline QnA
- **Stripe** for future business purpose (if I have users)
- **Copilot AI** for replies and composing
- **Orama** for powerful full text search
